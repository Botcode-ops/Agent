# AI Agents From Scratch

A hands-on, educational repository demonstrating how to build and orchestrate autonomous AI Agent systems from the ground up using Python and local LLMs via `llama-cpp-python`.
This repository serves as a step-by-step guide to understanding core agentic workflows, starting from basic chat interactions to advanced patterns like grammar-constrained structured actions, DAG dependency planners, self-evaluating reliability loops, tracing, and multi-agent coordination.
